# Task: Analisar Canal

Workflow guiado para espionar e analisar um canal/perfil.

---

## Pré-requisitos

- [ ] Link ou @ do canal/perfil
- [ ] Plataforma definida (YouTube, Instagram, TikTok)
- [ ] Objetivo da análise claro (hooks, thumbnails, temas, tudo)

---

## Passo a Passo

### 1. Coletar Informações

```
Responda:
1. Qual o canal/perfil? (link ou @)
2. Qual plataforma? (YouTube, Instagram, TikTok)
3. O que você quer descobrir?
   - [ ] Top vídeos/posts
   - [ ] Padrões de hook
   - [ ] Thumbnails que funcionam
   - [ ] Temas em alta
   - [ ] Frequência de posts
   - [ ] Tudo acima
4. Qual seu nicho? (pra eu conectar as descobertas com você)
```

### 2. Coletar Dados do Canal

**Para YouTube:**
```
1. Acessar canal
2. Listar últimos 20-50 vídeos
3. Ordenar por views
4. Pegar: título, views, likes, duração, data
5. Se possível, pegar transcrições dos top 5
```

**Para Instagram:**
```
1. Acessar perfil
2. Listar últimos 20-30 posts/reels
3. Identificar os com mais engajamento
4. Pegar: tipo (reel/carrossel/estático), likes, comentários
```

**Para TikTok:**
```
1. Acessar perfil
2. Listar vídeos
3. Ordenar por views/engajamento
4. Pegar: views, likes, comentários
```

### 3. Analisar Top Performers

```
1. Selecionar top 10 por performance
2. Para cada um, analisar:
   - O que tem de especial?
   - Como abre? (hook)
   - Qual o tema?
   - Qual a estrutura?
   - Thumbnail (se YouTube)
   - Duração
```

### 4. Identificar Padrões

```
HOOKS:
- Como os top vídeos abrem?
- Fórmula comum?
- Palavras que repetem?

THUMBNAILS (YouTube):
- Cores dominantes?
- Texto na thumb?
- Expressão facial?
- Elementos visuais?

TEMAS:
- Quais assuntos performam melhor?
- Há temas que sempre funcionam?
- Há temas que nunca funcionam?

TÍTULOS:
- Estrutura comum?
- Palavras-chave?
- Comprimento?

FREQUÊNCIA:
- Quantos posts/semana?
- Dias específicos?
- Horários?
```

### 5. Extrair Insights Acionáveis

```
1. O que você pode replicar?
2. O que adaptar pro seu contexto?
3. O que evitar?
4. Próximos passos práticos
```

### 6. Entregar Relatório
