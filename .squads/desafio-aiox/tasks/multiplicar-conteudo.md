# Task: Multiplicar Conteúdo

Workflow guiado para transformar 1 peça de conteúdo em múltiplos formatos.

---

## Pré-requisitos

- [ ] Conteúdo original (vídeo, texto, áudio)
- [ ] Plataformas destino definidas
- [ ] Quantidade desejada por formato
- [ ] Tom de voz definido

---

## Passo a Passo

### 1. Coletar Informações

```
Responda:
1. Qual o conteúdo original? (link ou texto)
2. Tipo do conteúdo? (vídeo, podcast, blog, newsletter)
3. Duração/tamanho?
4. Tema principal?
5. Para quais plataformas?
   - [ ] Instagram Reels
   - [ ] Instagram Carrossel
   - [ ] Instagram Stories
   - [ ] YouTube Shorts
   - [ ] TikTok
   - [ ] X/Twitter (thread)
   - [ ] LinkedIn
   - [ ] Newsletter
   - [ ] Blog
6. Quantidade por formato?
7. Tom de voz? (educativo, inspiracional, provocador, leve)
```

### 2. Consumir e Mapear

```
1. Ler/assistir conteúdo completo
2. Identificar pontos-chave:
   - Insights principais
   - Dicas práticas
   - Histórias/exemplos
   - Frases de impacto
   - Estatísticas/dados
3. Dividir em fragmentos independentes
4. Cada fragmento = 1 potencial peça
```

### 3. Matriz de Multiplicação

```
CONTEÚDO ORIGINAL: [descrever]

FRAGMENTOS IDENTIFICADOS:
1. [Fragmento 1 - tema]
2. [Fragmento 2 - tema]
3. [Fragmento 3 - tema]
...

DISTRIBUIÇÃO:
- Fragmento 1 → Reel + Post estático + Tweet
- Fragmento 2 → Carrossel + Thread
- Fragmento 3 → Short + Story
...
```

### 4. Criar Cada Peça

Para cada formato, seguir estrutura específica:

---

#### Reels/Shorts (60s)

```markdown
## Reel: [Título]

**Baseado em:** Fragmento #X

**Hook (0-3s):**
"[Frase de abertura]"

**Desenvolvimento (3-55s):**
[Pontos principais]

**CTA (55-60s):**
"[Call to action]"

**Texto na tela:**
[Sugestões]

**Legenda:**
[Pronta com hashtags]
```

---

#### Carrossel (7-10 slides)

```markdown
## Carrossel: [Título]

**Baseado em:** Fragmento #X

**Slide 1 (Capa):**
[Headline que prende]

**Slides 2-6:**
[Conteúdo slide a slide]

**Slide 7 (CTA):**
[Call to action]

**Legenda:**
[Pronta com hashtags]
```

---

#### Thread X/Twitter

```markdown
## Thread: [Tema]

**Baseado em:** Fragmento #X

**Tweet 1 (Hook):**
[Max 280 chars]

**Tweets 2-7:**
2/ [ponto]
3/ [ponto]
...

**Tweet Final:**
[CTA]
```

---

#### Post LinkedIn

```markdown
## LinkedIn: [Tema]

**Baseado em:** Fragmento #X

**Abertura (hook):**
[Primeira linha que aparece antes do "ver mais"]

**Corpo:**
[Conteúdo - max 1300 chars]

**CTA:**
[Call to action]

**Hashtags:**
[3-5 hashtags]
```

---

#### Newsletter

```markdown
## Newsletter: [Assunto]

**Baseado em:** [Todo conteúdo ou fragmentos X, Y, Z]

**Subject:**
[Título do email]

**Preview:**
[Texto de preview]

**Corpo:**
[Conteúdo formatado pra email]

**CTA:**
[Call to action]
```

---

## Checklist Final

- [ ] Cada peça faz sentido sozinha?
- [ ] Hook forte em todas?
- [ ] CTA em todas?
- [ ] Tom consistente?

- [ ] Quantidade pedida entregue?
- [ ] Organizado e pronto pra usar?
