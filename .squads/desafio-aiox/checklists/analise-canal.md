# Checklist: Analise de Canal

Use este checklist para validar analises de canais/perfis antes de entregar.

---

## Coleta de Dados

### Informacoes Basicas
- [ ] **Nome do canal/perfil correto?**
- [ ] **Plataforma identificada?** (YouTube, Instagram, TikTok)
- [ ] **Numero de seguidores/inscritos?**
- [ ] **Total de videos/posts?**
- [ ] **Data da analise registrada?**

### Metricas Coletadas
- [ ] **Views/visualizacoes?**
- [ ] **Likes/curtidas?**
- [ ] **Comentarios?**
- [ ] **Shares (se disponivel)?**
- [ ] **Frequencia de posts?**

---

## Top Performers

### Identificacao
- [ ] **Top 10 videos/posts listados?**
- [ ] **Ordenados por performance (views/engajamento)?**
- [ ] **Metricas de cada um incluidas?**
- [ ] **Links/referencias incluidos?**

### Analise de Cada Top
- [ ] **Titulo/headline analisado?**
- [ ] **Duracao/tamanho registrado?**
- [ ] **Data de publicacao?**
- [ ] **Por que performou bem? (hipotese)**

---

## Padroes Identificados

### Hooks
- [ ] **Padroes de abertura identificados?**
- [ ] **Exemplos especificos listados?**
- [ ] **Formulas extraidas?**
- [ ] **O que NAO funciona identificado?**

### Thumbnails (YouTube)
- [ ] **Cores dominantes identificadas?**
- [ ] **Uso de texto analisado?**
- [ ] **Expressoes faciais mapeadas?**
- [ ] **Elementos visuais recorrentes?**
- [ ] **Formula de thumbnail extraida?**

### Titulos
- [ ] **Estrutura comum identificada?**
- [ ] **Palavras-chave recorrentes?**
- [ ] **Comprimento medio?**
- [ ] **Formulas de titulo extraidas?**

### Temas
- [ ] **Temas que performam bem listados?**
- [ ] **Temas que NAO performam identificados?**
- [ ] **Tendencias do nicho mapeadas?**

---

## Insights Acionaveis

### Recomendacoes
- [ ] **Minimo 3 recomendacoes praticas?**
- [ ] **Conectadas com o nicho do usuario?**
- [ ] **Especificas o suficiente pra executar?**
- [ ] **Priorizadas por impacto?**

### O que Replicar
- [ ] **Lista do que o usuario pode copiar?**
- [ ] **Adaptado pro contexto dele?**
- [ ] **Realista de executar?**

### O que Evitar
- [ ] **Lista do que NAO fazer?**
- [ ] **Baseado em evidencia (nao achismo)?**

---

## Proximos Passos

- [ ] **Acoes claras e especificas?**
- [ ] **Ordenadas por prioridade?**
- [ ] **Prazo/urgencia indicados?**
- [ ] **Responsavel definido (se aplicavel)?**

---

## Qualidade do Relatorio

### Estrutura
- [ ] **Resumo executivo no inicio?**
- [ ] **Secoes bem organizadas?**
- [ ] **Facil de ler e navegar?**
- [ ] **Tabelas quando apropriado?**

### Profundidade
- [ ] **Analise vai alem do obvio?**
- [ ] **Insights sao valiosos (nao genericos)?**
- [ ] **Dados suportam conclusoes?**
- [ ] **Comparativos uteis (se aplicavel)?**

### Precisao
- [ ] **Dados verificados?**
- [ ] **Numeros corretos?**
- [ ] **Links funcionam?**
- [ ] **Sem suposicoes sem base?**

|----------|------|-----|

**Resultado:**
- 90-100% → Excelente - Entregar
- 75-89% → Bom - Ajustes menores
- 60-74% → Regular - Completar lacunas
- <60% → Insuficiente - Refazer

---

## Erros Comuns

```
COLETA
- Pegar so metricas de views, ignorar engajamento
- Nao verificar se dados estao atualizados
- Confundir total de views com views recentes

ANALISE
- Conclusoes genericas ("ele faz videos bons")
- Nao explicar POR QUE algo funciona
- Ignorar o que NAO funciona
- Nao adaptar pro contexto do usuario

ENTREGA
- Relatorio muito longo sem resumo
- Insights nao acionaveis
- Sem proximos passos claros
```

## Benchmarks

### Por Tipo de Analise

| Tipo | Tempo Estimado | Profundidade |
|------|----------------|--------------|
| Quick Check | 15-30 min | Top 5, padroes basicos |
| Analise Padrao | 1-2h | Top 10, hooks, thumbnails, temas |
| Deep Dive | 3-5h | Tudo acima + historico + comparativos |

### Por Plataforma

| Plataforma | Foco Principal | Metricas-Chave |
|------------|----------------|----------------|
| YouTube | Thumbnails, titulos, retencao | Views, Watch Time, CTR |
| Instagram | Hooks, estetica, legendas | Likes, Saves, Shares |
| TikTok | Hooks, trends, audio | Views, Completion Rate |
