# Checklist: Repurposing de Conteudo

Use este checklist para validar multiplicacao de conteudo antes de entregar.

---

## Conteudo Original

### Analise Inicial
- [ ] **Conteudo original identificado?** (link, transcricao, texto)
- [ ] **Tipo definido?** (video, podcast, blog, newsletter)
- [ ] **Duracao/tamanho registrado?**
- [ ] **Tema principal claro?**
- [ ] **Publico-alvo definido?**

### Fragmentacao
- [ ] **Conteudo dividido em fragmentos?**
- [ ] **Cada fragmento faz sentido sozinho?**
- [ ] **Pontos-chave identificados?**
- [ ] **Quotes de impacto extraidas?**
- [ ] **Dados/estatisticas separados?**

---

## Distribuicao por Plataforma

### Plataformas Solicitadas
- [ ] **Instagram Reels?**
- [ ] **Instagram Carrossel?**
- [ ] **Instagram Stories?**
- [ ] **YouTube Shorts?**
- [ ] **TikTok?**
- [ ] **X/Twitter (thread)?**
- [ ] **LinkedIn?**
- [ ] **Newsletter?**
- [ ] **Blog?**

### Quantidade
- [ ] **Quantidade por formato atende o pedido?**
- [ ] **Distribuicao faz sentido?** (nao repetir mesmo conteudo)

## Qualidade por Formato

### Reels/Shorts (Cada um)
- [ ] Hook nos primeiros 3s?
- [ ] Desenvolvimento claro?
- [ ] CTA no final?
- [ ] Duracao adequada (30-60s)?
- [ ] Texto na tela sugerido?
- [ ] Legenda com hashtags?

### Carrossel (Cada um)
- [ ] Capa com headline forte?
- [ ] 5-7 slides de conteudo?
- [ ] Slide final com CTA?
- [ ] Fluxo logico entre slides?
- [ ] Legenda complementar (nao repete)?
- [ ] Hashtags relevantes?

### Thread X/Twitter
- [ ] Tweet 1 e hook forte?
- [ ] Cada tweet agrega valor?
- [ ] Numeracao correta (1/, 2/, 3/)?
- [ ] Tweet final com CTA?
- [ ] Respeita limite de 280 chars?

### Post LinkedIn
- [ ] Primeira linha prende atencao?
- [ ] Tom profissional mas humano?
- [ ] Valor > promocao?
- [ ] CTA claro?
- [ ] 3-5 hashtags relevantes?

### Newsletter
- [ ] Subject line atrativa?
- [ ] Preview text funciona?
- [ ] Abertura pessoal?
- [ ] Conteudo formatado pra email?
- [ ] CTA claro?

## Adaptacao de Linguagem

### Por Plataforma
- [ ] **Tom adaptado pra cada rede?**
  - Instagram: visual, leve, emojis OK
  - LinkedIn: profissional, insights
  - X/Twitter: direto, provocador
  - TikTok: autentico, trends

### Consistencia
- [ ] **Mensagem central mantida?**
- [ ] **Nao distorce o original?**
- [ ] **Voz do criador preservada?**

## Organizacao da Entrega

### Estrutura
- [ ] **Organizado por plataforma?**
- [ ] **Cada peca numerada/nomeada?**
- [ ] **Facil de encontrar?**
- [ ] **Pronto pra copiar/colar?**

### Calendario Sugerido
- [ ] **Sugestao de quando postar?**
- [ ] **Espacamento faz sentido?**
- [ ] **Considera melhor horario por rede?**

### Resumo
- [ ] **Total de pecas listado?**
- [ ] **Breakdown por formato?**
- [ ] **Estimativa de tempo de conteudo?**

## Matriz de Multiplicacao (Referencia)

### De Video Longo (10-15 min)
```
→ 5-8 Reels/Shorts (60s cada)
→ 2-4 Carrosseis (7 slides)
→ 1-2 Threads (8 tweets)
→ 5-10 Posts estaticos (quotes)
→ 1 Newsletter
→ 1 Blog post
→ 10-15 Stories

TOTAL: 25-40 pecas
```

### De Live (1-2h)
```
→ 10-20 Reels/Shorts
→ 3-5 Carrosseis
→ 2-3 Threads
→ 10-20 Posts estaticos
→ 2-3 Newsletters
→ 1-2 Blog posts

TOTAL: 30-50 pecas
```

### De Podcast (1h)
```
→ 8-12 Audiogramas/Clips
→ 3-5 Carrosseis
→ 1-2 Threads
→ 10-15 Tweets
→ 1 Newsletter
→ 1 Blog post

TOTAL: 25-35 pecas
```

## Erros Comuns

```
FRAGMENTACAO
- Fragmentos que precisam de contexto do original
- Fragmentos muito longos (nao cabem no formato)
- Ignorar quotes de impacto

ADAPTACAO
- Usar mesmo texto pra todas as plataformas
- Tom muito formal pro Instagram
- Tom muito casual pro LinkedIn
- Ignorar cultura de cada rede

HOOKS
- Hooks fracos ou genericos
- Copiar hook do original sem adaptar
- Hooks que nao funcionam no formato curto

CTAs
- Mesmo CTA em tudo
- CTA generico ("segue ai")
- Esquecer CTA

ENTREGA
- Desorganizado, dificil de navegar
- Sem numeracao/nomes claros
- Sem resumo do total
```
