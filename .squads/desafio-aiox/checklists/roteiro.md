# Checklist: Roteiro de Video

Use este checklist para validar roteiros antes de entregar.

---

## Briefing

### Informacoes Recebidas
- [ ] **Tipo de roteiro definido?** (short, reel, video longo, VSL, ad)
- [ ] **Tema/assunto claro?**
- [ ] **Duracao definida?**
- [ ] **Objetivo claro?** (educar, entreter, vender, engajar)
- [ ] **Publico-alvo definido?**
- [ ] **Tom de voz definido?** (serio, leve, provocador, inspiracional)
- [ ] **CTA desejado definido?**

---

## Hook (Primeiros 3 Segundos)

### Impacto
- [ ] **Prende atencao imediatamente?**
- [ ] **Nao comeca com "Oi gente" ou apresentacao?**
- [ ] **Gera curiosidade ou emocao?**
- [ ] **Promete algo especifico?**
- [ ] **Funciona sem contexto anterior?**

### Tipo de Hook (marcar qual)
- [ ] Pergunta provocativa ("Voce sabia que...?")
- [ ] Afirmacao chocante ("Isso destruiu meu negocio")
- [ ] Promessa clara ("Em 60s voce vai aprender...")
- [ ] Provocacao ("Se voce faz isso, esta perdendo dinheiro")
- [ ] Historia ("Ha 2 anos eu estava quebrado...")
- [ ] Numero ("3 erros que todo iniciante comete")
- [ ] Contraintuitivo ("Pare de fazer X se quer Y")

```
Score: ___/10 (meta: >= 7)
```

## Desenvolvimento (Corpo)

### Estrutura
- [ ] **Fluxo logico?** (começo → meio → fim)
- [ ] **Pontos bem definidos?** (1, 2, 3...)
- [ ] **Cada ponto agrega valor?**
- [ ] **Sem enrolacao?**
- [ ] **Ritmo adequado?**

### Conteudo
- [ ] **Entrega o que o hook prometeu?**
- [ ] **Informacao util/valiosa?**
- [ ] **Exemplos praticos incluidos?**
- [ ] **Faz sentido pra quem nao conhece o criador?**

### Para Videos Longos
- [ ] **Secoes bem divididas?**
- [ ] **Introducao breve?**
- [ ] **Recapitulacao antes do CTA?**
- [ ] **B-roll/cortes sugeridos?**

### Para VSL
- [ ] **Problema aprofundado?**
- [ ] **Solucao apresentada claramente?**
- [ ] **Prova social incluida?**
- [ ] **Oferta estruturada?**
- [ ] **Objeccoes respondidas?**

---

## CTA (Call to Action)

### Estrutura
- [ ] **CTA claro e especifico?**
- [ ] **Conectado com o conteudo?**
- [ ] **Facil de executar?**
- [ ] **Apenas UMA acao principal?**

### Tipo de CTA (marcar qual)
- [ ] "Segue pra mais dicas como essa"
- [ ] "Salva esse video pra ver depois"
- [ ] "Comenta aqui [pergunta especifica]"
- [ ] "Manda pra alguem que precisa disso"
- [ ] "Link na bio" (se venda)
- [ ] "Se inscreve e ativa o sininho" (YouTube)
- [ ] Outro: _______________

### Qualidade do CTA
```
Score: ___/5 (meta: >= 4)
```

### Tom
- [ ] **Consistente do inicio ao fim?**
- [ ] **Natural (como conversa)?**
- [ ] **Adequado ao publico?**
- [ ] **Adequado a plataforma?**

### Clareza
- [ ] **Frases curtas e diretas?**
- [ ] **Sem jargao desnecessario?**
- [ ] **Facil de entender na primeira vez?**
- [ ] **Sem repeticoes desnecessarias?**

## Erros Comuns

```
HOOK
- Comeca com "Oi gente, bem-vindos..."
- Hook demora >3s pra chegar
- Promete algo que nao entrega
- Generico demais ("Vou te ensinar algo importante")

DESENVOLVIMENTO
- Enrolacao, fala muito sem dizer nada
- Pula pontos sem explicar
- Muito tecnico pro publico
- Nao usa exemplos praticos

CTA
- Esquece de colocar CTA
- CTA generico ("segue ai")
- Multiplos CTAs (confunde)
- CTA desconectado do conteudo

LINGUAGEM
- Tom inconsistente
- Muito formal pra formato curto
- Frases longas demais
- Repete as mesmas palavras
```
