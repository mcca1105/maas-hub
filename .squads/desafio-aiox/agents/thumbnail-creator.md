## O que e a Capa Lendaria?

**Aplicacao web** que gera thumbnails profissionais com IA em 3 formatos simultaneos:

| Formato | Tamanho | Uso |
|---------|---------|-----|
| **YouTube Thumbnail** | 16:9 (1280x720) | Capas de videos YouTube |
| **Reels/Shorts** | 9:16 (1080x1920) | Capas verticais pra Reels, Shorts, TikTok |
| **Feed Portrait** | 1080x1350 | Posts de feed Instagram |

**Versao atual:** V27.0 - Protocolo Realismo Factual
**Repositorio:** github.com/ericasouza-eng/DESAFIO-AIOX

## Como Usar (Passo a Passo)

### Passo 1: Escolher Base Facial

```
1. Clique em uma das fotos pre-carregadas
   OU
2. Clique em "+ ADD" pra fazer upload de nova foto

Dica: Use foto de alta qualidade com boa iluminacao
```

### Passo 2: Definir o Tema

```
1. No campo "TEMA", digite UMA palavra
2. Escolha algo FISICO que represente o assunto

Exemplos:
- Video sobre codigo → CODIGOS
- Video sobre dinheiro → DINHEIRO
- Video sobre IA → IA
- Video sobre produtividade → RELOGIO
- Video sobre vendas → VENDAS
```

**IMPORTANTE:** O tema vira um OBJETO REAL na cena. A IA cria elementos visuais 3D baseados nessa palavra.

### Passo 3: Descrever a Cena

```
1. No campo "CENA", descreva como quer o visual
2. Seja especifico mas conciso

Exemplos:
- "Fundo tecnologico com hologramas e luzes neon"
- "Ambiente executivo com tons dourados"
- "Cenario futurista com elementos de codigo"
```

### Passo 4: Adicionar Headline (Opcional)

```
1. No campo "HEADLINE", digite o texto que vai aparecer
2. Escolha a cor: BRANCO | DOURADO | VERMELHO

Exemplos de Headlines:
- "5 ERROS FATAIS"
- "R$10K/MES"
- "PARE AGORA"
- "CODIGOS"
```

### Passo 5: Gerar

```
1. Clique em "MATERIALIZAR 3 CAPAS"
2. Aguarde a geracao (30-60 segundos)
3. Receba 9 imagens:
   - 3 YouTube (16:9)
   - 3 Reels/Shorts (9:16)
   - 3 Feed (1080x1350)
4. Baixe as que gostar
```

## Workflow Recomendado

```
1. PREPARAR
   - Ter foto base de boa qualidade
   - Definir tema do video
   - Pensar na headline

2. GERAR
   - Rodar 2-3x com variacoes de tema/headline
   - Total: 18-27 opcoes

3. SELECIONAR
   - Escolher as 3 melhores
   - Uma pra cada plataforma

4. TESTAR
   - A/B test com 2 thumbnails diferentes
   - Ver qual tem mais CTR
```

## Troubleshooting

### Problema: Imagem muito generica
**Solucao:** Seja mais especifico na CENA. Adicione detalhes visuais.

### Problema: Headline ilegivel
**Solucao:** Use headline mais curta (2-3 palavras max). Escolha cor com contraste.

### Problema: Tema nao aparece
**Solucao:** Use palavra mais concreta. "RELOGIO" funciona melhor que "TEMPO".

### Problema: Rosto distorcido
**Solucao:** Use foto base de melhor qualidade. Frente pro rosto, boa iluminacao.
