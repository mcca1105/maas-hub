### 🐍 **5 Scripts Python de Validação** (`/scripts/`)

- **`validate-copy-quality.py`** — Score /100; verifica especificidade (Gary Halbert), abertura proibida, elementos estruturais, voz passiva, balanceamento você/nós
- **`score-awareness-stage.py`** — Diagnostica estágio Eugene Schwartz do copy (1-5); analisa VOC para descobrir estágio real da audiência; detecta mismatch
- **`drm-cohesion-validator.py`** — Valida message match, preço, vocabulário e urgência entre todos os assets do funil
- **`validate-rmbc-brief.py`** — Valida brief RMBC (Stefan Georgi): Research completeness, Mechanism naming, Brief 7 componentes, hook count; BLOQUEIA escrita prematura
- **`validate-cialdini-audit.py`** — Audita 7 Princípios de Cialdini no copy; modo `--strict` flagra fake scarcity e padrões antiéticos

```bash
# Exemplos de uso:
python scripts/validate-copy-quality.py --file tasks/write-vsl-script.md
python scripts/score-awareness-stage.py --copy vsl.md --research voc-data.md
python scripts/drm-cohesion-validator.py --dir funnel-assets/
python scripts/validate-rmbc-brief.py --file brief.md
python scripts/validate-cialdini-audit.py --file sales-letter.md --strict
```
