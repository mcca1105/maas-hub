# Checklist: Webinar Script Quality

**Use when:** Reviewing a webinar script before going live or recording evergreen.

---

## Section 1: Structure & Timing

- [ ] Content section is at least 50% of total webinar length
- [ ] Intro section is 10-15 minutes maximum (not 30 minutes)
- [ ] Offer section starts no later than 75% of total duration
- [ ] Q&A/close section included (minimum 10 minutes)
- [ ] Interactive elements included in every lesson (poll/exercise/chat question)

## Section 2: Opening Hook

- [ ] First spoken words are the hook (no "Can everyone hear me? Let me know in the chat")
- [ ] Core promise is stated within the first 3 minutes
- [ ] Overview of content is previewed (tell them what they'll learn)
- [ ] Credibility is established through story, not credential list
- [ ] Audience is told why this matters NOW (urgency of topic)

## Section 3: Teaching Content

- [ ] Each lesson has a clear title and single learning objective
- [ ] Teaching is specific (includes numbers, names, examples, not vague principles)
- [ ] At least one "common mistake" section per lesson
- [ ] Each lesson ends with an interactive element (exercise or chat question)
- [ ] Lessons build logically — each one sets up the need for the next

## Section 4: Transition to Offer

- [ ] Transition doesn't feel abrupt or mechanical
- [ ] Teaching content is acknowledged as genuinely valuable ("you could implement this yourself")
- [ ] The offer is introduced as a natural extension, not a pivot
- [ ] The story of HOW the offer was created is included (credibility)
- [ ] Offer is clearly positioned for the specific audience attending

## Section 5: Offer Presentation

- [ ] Value stack is presented before price (anchor first)
- [ ] Each component in value stack has stated dollar value
- [ ] Bonuses are time-limited (genuine deadline or quantity limit)
- [ ] Price reveal follows: "you won't pay X, not even half, today it's just Y" structure
- [ ] Guarantee is specific and trust-building

## Section 6: Q&A / Objection Handling

- [ ] Top 5 objections are pre-scripted (don't wait for questions — anticipate them)
- [ ] Price objection handled with value reframe and cost of inaction
- [ ] Time objection handled with minimum viable use approach
- [ ] "Does this work for me?" objection handled with identity-matching testimonial
- [ ] Offer is re-stated clearly before webinar ends

## Section 7: Conversational Quality

- [ ] Script sounds like spoken conversation (not "presented prose")
- [ ] Run the script at 125-150 wpm reading speed — does timing work?
- [ ] "You" appears more than "we/I" throughout content section
- [ ] Personal story/vulnerability shared at least once (Frank Kern rule)
- [ ] Humor or levity present (even serious content needs a moment)

## Veto Conditions

- VETO if content section is under 40% of total duration
- VETO if credibility is a list of accomplishments (needs story, not resume)
- VETO if no interactive elements in content section
- VETO if guarantee is vague ("satisfaction guaranteed")
- VETO if no Q&A objection handling section
