## 🔴 CRITICAL — Brief Completeness Gate

The Brief must answer every question in this section before one word of copy is written.

### Prospect
- [ ] Target avatar is named and described in 1 paragraph
- [ ] Awareness stage confirmed: Stage _____
- [ ] Traffic temperature confirmed: Cold / Warm / Hot
- [ ] Dominant emotion identified: ___________________
- [ ] Top 3 pains documented (in their exact language)
- [ ] Dream outcome documented (specific and vivid)
- [ ] Top 3 objections to purchasing documented

### Mechanism
- [ ] Mechanism has a proprietary name: ___________________
- [ ] One-sentence mechanism explanation written (30 words max):
  ```
  _____________________________________________________________
  ```
- [ ] Primary proof supporting the mechanism identified: ___________________
- [ ] "Villain" (root cause the mechanism addresses) named: ___________________

### Headlines
- [ ] Minimum 10 headline options written
- [ ] Top 3 starred for selection
- [ ] Each headline matches the awareness stage of the prospect
- [ ] Control headline selected: ___________________

### Story / Hook
- [ ] Opening story or hook type selected:
  - [ ] Founder story
  - [ ] Customer transformation story
  - [ ] Research discovery / news format
  - [ ] Pattern interrupt / counterintuitive claim
- [ ] Protagonist of the story identified (who is it about?): ___________________
- [ ] The "before" state of the protagonist maps to the prospect's situation: YES / NO
- [ ] Discovery moment / turning point identified: ___________________

### Offer
- [ ] Core product name: ___________________
- [ ] Price confirmed: $___________________
- [ ] All bonuses listed with names and standalone value:
  - Bonus 1: _______________ — Value: $___
  - Bonus 2: _______________ — Value: $___
  - Bonus 3: _______________ — Value: $___
- [ ] Total value stack: $___________________
- [ ] Guarantee terms: ___________________
- [ ] Urgency/scarcity element: ___________________

### Proof Stack
- [ ] Number of testimonials available: ___
- [ ] Best 3 testimonials identified (most specific, most relevant):
  1. [Name] — [Result] in [Timeframe]
  2. [Name] — [Result] in [Timeframe]
  3. [Name] — [Result] in [Timeframe]
- [ ] Clinical/scientific data available: YES / NO
- [ ] Before-and-after documentation: YES / NO

## 🟡 HIGH PRIORITY — Copy Structure Pre-Decision

### Asset Format Confirmed
- [ ] What are we writing?
  - [ ] VSL script (for video)
  - [ ] Long-form sales letter (for web or print)
  - [ ] Short-form sales page
  - [ ] Email sales sequence
  - [ ] Other: ___________________
- [ ] Optimal length for this format and audience confirmed

### Structural Sequence
- [ ] Open: Pattern interrupt? Story? Direct claim? Type selected: ___________________
- [ ] Story arc mapped: Before → Discovery → After
- [ ] Mechanism revelation position identified (not too early!)
- [ ] Offer placement: After mechanism and proof stack (not before)
- [ ] CTA count and placement defined:
  - CTA 1 position: ___________________
  - CTA 2 position: ___________________
  - Final CTA: End of copy / P.S.

## 🟢 BEST PRACTICE — Pre-Write Confidence

Before starting copy, answer these three questions out loud:

**1. Who am I talking to and what are they feeling right now?**
```
_______________________________________________________________
```

**2. What is the ONE thing I need them to believe by the end of this piece?**
```
_______________________________________________________________
```

**3. What is the most dramatic, specific result I can promise and prove?**
```
_______________________________________________________________
```

If you can't answer all three fluently, the Brief is not complete.

---

## ❌ AUTO-FAIL CONDITIONS

- [ ] ❌ No proprietary mechanism name (generic = low conversion)
- [ ] ❌ Fewer than 10 headline options (selected too early = sub-optimal headline)
- [ ] ❌ Offer details incomplete (price, bonuses, guarantee all needed before writing)
- [ ] ❌ No testimonials identified (copy will need to over-rely on claims)
- [ ] ❌ Story/protagonist not selected (opening will be unfocused)
- [ ] ❌ Objections not listed (they'll derail the sale if unaddressed)
