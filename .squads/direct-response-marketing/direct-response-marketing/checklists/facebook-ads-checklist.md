# Checklist: Facebook/Instagram Ad Quality

**Use when:** Reviewing Facebook or Instagram ad copy before launch.

---

## Section 1: Primary Text (The Copy)

- [ ] First line is the hook — no warm-up sentences before it
- [ ] First 125 characters work standalone (visible before "See More")
- [ ] First 125 characters create curiosity OR identify the problem OR interrupt the pattern
- [ ] Copy does NOT open with company name or "Hi, I'm [name]"
- [ ] Body expands on hook naturally (doesn't feel like a different conversation)
- [ ] CTA line is specific ("Click the link below and download your free guide" not "Learn more")

## Section 2: Headline (Below Image)

- [ ] Headline is under 40 characters (hard limit — gets truncated if over)
- [ ] Headline contains the key benefit or mechanism
- [ ] Headline stands alone without reading the primary text
- [ ] Headline is benefit-focused, not clever-but-vague

## Section 3: Ad Variations

- [ ] At least 3 ad variations written
- [ ] Each variation tests ONE different variable (different angle OR different hook type)
- [ ] All 3 variations have different hooks (not same hook, different words)
- [ ] At least one variation uses social proof as the lead
- [ ] At least one variation uses curiosity or story as the lead

## Section 4: Traffic Temperature Calibration

- [ ] Cold traffic: Ad doesn't assume prior knowledge of brand/product
- [ ] Cold traffic: Ad leads with problem identification or curiosity (not price/offer)
- [ ] Warm traffic: Ad references the angle they've seen (continuity)
- [ ] Retargeting: Ad addresses likely reason they didn't convert

## Section 5: Visual Direction

- [ ] Specific visual direction written for each ad variation
- [ ] Visual is not a generic stock photo
- [ ] Visual creates curiosity or shows result (not product-on-white-background for cold traffic)
- [ ] Visual is appropriate for the platform placement (square for feed, vertical for Stories)

## Section 6: Technical Compliance

- [ ] CTA button selection matches campaign objective and destination
- [ ] Ad doesn't make prohibited claims (platform-specific rules checked)
- [ ] Story/Reels version adapted (shorter copy for these placements)
- [ ] Character counts verified: Primary (125 visible), Headline (40), Description (30)

## Section 7: Testing Setup

- [ ] A/B test structure defined before launch
- [ ] UTM parameters added to destination URL
- [ ] Conversion event confirmed as tracking correctly
- [ ] Kill threshold defined (when to pause underperforming ads)

## Veto Conditions

- VETO if headline exceeds 40 characters
- VETO if all 3 variations use the same angle
- VETO if "Shop Now" used for non-e-commerce destination
- VETO if visual direction is "use a relevant image" (too vague)
- VETO if no testing structure defined
