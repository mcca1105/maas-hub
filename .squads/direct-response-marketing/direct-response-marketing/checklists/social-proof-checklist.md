# Social Proof Checklist
**Agent:** `@drm:robert-cialdini`
**Purpose:** Validate social proof quality, placement, and persuasion power
**Standard:** Robert Cialdini — Influence (Social Proof Principle)
**Run:** Before publishing any sales asset or funnel page

---

## 🔴 CRITICAL — Proof Quality

### The Specificity Test
Every testimonial must pass this test:
- [ ] Contains a SPECIFIC result (not "I loved it!" but "I lost 23 lbs in 6 weeks")
- [ ] Contains a timeframe (not "I improved" but "in 3 months")
- [ ] Contains a real name (first + last name, not "J. Smith" or "Anonymous")
- [ ] BONUS: Contains photo, video, or occupation context

**Audit each testimonial:**
| # | Name | Specific Result | Timeframe | Photo? | Pass/Fail |
|---|------|----------------|-----------|--------|-----------|
| 1 | | | | | |
| 2 | | | | | |
| 3 | | | | | |

### Testimonial Types Coverage
- [ ] At least 3 specific result testimonials (quantity + quality)
- [ ] At least 1 testimonial from a skeptic who changed their mind
- [ ] At least 1 testimonial that addresses the #1 objection
- [ ] At least 1 "like me" testimonial matching the target avatar

### Credibility Verification
- [ ] Every testimonial is from a real person (verifiable if challenged)
- [ ] Results described are achievable and not outliers presented as typical
- [ ] Disclaimers present where required by FTC/advertising standards
- [ ] No fabricated or incentivized testimonials (ethical and legal requirement)

---

## 🟡 HIGH PRIORITY — Proof Placement

### Strategic Positioning
- [ ] Social proof appears AFTER mechanism explanation (proof of concept)
- [ ] Social proof appears BEFORE offer reveal (reduces price resistance)
- [ ] Social proof appears NEAR the CTA (last-moment reassurance)
- [ ] Proof is absent from the very opening (don't lead with it — earn it first)

### Proof Density by Funnel Step
- [ ] Sales page: minimum 5-10 testimonials
- [ ] Opt-in/landing page: 2-3 testimonials acceptable
- [ ] Email sequence: 1 specific case study per conversion email
- [ ] Ad creative: 1 strong testimonial (if using social proof format)

### Authority Proof (if applicable)
- [ ] Expert endorsements present (if available)
- [ ] Media mentions displayed ("As seen in...")
- [ ] Clinical/scientific data cited with source
- [ ] Awards or certifications displayed (relevant ones only)

---

## 🟢 BEST PRACTICE — Proof Stacking

### Types of Proof to Combine
Stack multiple proof types for maximum impact:
- [ ] Customer testimonials (most relatable)
- [ ] Case studies (most detailed)
- [ ] Data/statistics (most credible)
- [ ] Expert endorsements (borrowed authority)
- [ ] Media coverage (third-party validation)
- [ ] User numbers ("47,000 customers in 38 countries")
- [ ] Before/after documentation (most visual)

### "Like Me" Proof
Highest-converting testimonials are from people who match the prospect:
- [ ] Testimonials represent the age range of target audience
- [ ] Testimonials represent the situation/problem of target audience
- [ ] If multiple avatar segments exist, proof for each is available
- [ ] Geographic diversity if selling internationally

### Video Testimonials
- [ ] Video testimonials available (convert significantly higher than text)
- [ ] Videos are under 90 seconds (longer = lower completion)
- [ ] Videos are candid and natural (not over-produced/scripted)
- [ ] Captions available on all video testimonials

---

## ❌ AUTO-FAIL CONDITIONS

- [ ] ❌ Any testimonial says only "Great product!" with no specific result
- [ ] ❌ Testimonials are anonymous or use only first names
- [ ] ❌ Results described are impossibly extreme and not typical
- [ ] ❌ No disclaimer when results are exceptional ("results not typical" / "results vary")
- [ ] ❌ All testimonials from the same demographic (not representative)
- [ ] ❌ Testimonials placed only at the bottom (no proof at moments of doubt)
- [ ] ❌ No testimonials at all on a high-ticket offer page

## TESTIMONIAL COLLECTION SYSTEM

When gathering new testimonials, ask customers:
```
TESTIMONIAL REQUEST TEMPLATE

Hi [Name],

We'd love to share your results with others who might benefit.
Can you help us by answering 3 quick questions?

1. What was your situation BEFORE using [product]?
   (Be as specific as possible)

2. What specific results have you gotten since?
   (Numbers, timeframes, and specifics are most helpful)

3. What would you tell someone who's on the fence about trying it?

May we use your response and first/last name? [ ] Yes [ ] Yes, but use [alternative]
Photo (optional): [ ] Happy to share a photo
```
