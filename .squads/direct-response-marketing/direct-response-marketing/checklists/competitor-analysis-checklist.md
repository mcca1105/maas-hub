# Competitor Analysis Checklist
**Agent:** `@drm:eugene-schwartz` / `@drm:stefan-georgi`
**Purpose:** Systematic competitive intelligence for copy and offer positioning
**Standard:** Schwartz Market Sophistication + Georgi Research Phase
**Run:** At the start of any new market entry or before major copy rewrites

---

## 🔴 CRITICAL — Intelligence Gathering

### Competitor Identification
- [ ] Top 3-5 direct competitors identified (same product, same audience)
- [ ] Top 2-3 indirect competitors identified (different product, same problem solved)
- [ ] Competitor sources: Google Ads (spy tools), Facebook Ad Library, industry forums, affiliate networks

**Competitor list:**
```
1. _______________ | URL: _______________
2. _______________ | URL: _______________
3. _______________ | URL: _______________
4. _______________ | URL: _______________
5. _______________ | URL: _______________
```

### Core Intelligence Per Competitor
For EACH competitor, document:

**Mechanism/Big Idea:**
- [ ] What is their primary mechanism? (the "why it works" claim)
- [ ] Is their mechanism scientific, proprietary, or vague?
- [ ] Could we use a completely different mechanism and own our own lane?

**Offer Structure:**
- [ ] Price point: $___
- [ ] Bonuses included: Yes / No / List
- [ ] Guarantee: Yes / No / Terms
- [ ] Upsells visible: Yes / No / Description

**Copy Approach:**
- [ ] Headline type: Curiosity / Benefit / News / Command
- [ ] Traffic temperature targeted: Cold / Warm / Hot
- [ ] Apparent awareness stage targeted: 1 / 2 / 3 / 4 / 5
- [ ] Story format: Founder / Customer / Research / Other

---

## 🟡 HIGH PRIORITY — Positioning Intelligence

### Claims Analysis
- [ ] List every major claim each competitor makes
- [ ] Identify the most-repeated claim across all competitors (commoditized = avoid)
- [ ] Identify gaps — what NO competitor is saying but prospects want to hear

**Claims matrix:**
| Claim | Competitor 1 | Competitor 2 | Competitor 3 | Available to Us? |
|-------|-------------|-------------|-------------|-----------------|
| | | | | |
| | | | | |

### Mechanism Gap Analysis
Following Schwartz's sophistication framework:
- [ ] What mechanisms have already been used in this market?
- [ ] At what sophistication level is the market? (1-5)
- [ ] Is there a mechanism angle no competitor has used?
- [ ] White space identified: ___________________

### Proof Stack Analysis
- [ ] Competitor testimonial quality: High / Medium / Low
- [ ] Are competitors using case studies or just general testimonials?
- [ ] Do they have clinical/scientific proof?
- [ ] Can we out-prove them? How?

---

## 🟢 BEST PRACTICE — Positioning Decision

### Differentiation Strategy
Based on analysis, select positioning approach:
- [ ] **Different mechanism** — we explain WHY differently
- [ ] **Different audience** — we speak to a sub-segment competitors ignore
- [ ] **Different proof** — we have better/more specific results
- [ ] **Different offer** — better guarantee, more bonuses, better price
- [ ] **Different voice** — personality and style as the differentiator

### Weaknesses to Exploit
- [ ] Competitor #1 weakness identified: ___________________
- [ ] Competitor #2 weakness identified: ___________________
- [ ] Competitor #3 weakness identified: ___________________
- [ ] Copy addresses at least one of these weaknesses (without naming competitors)

### Ad Spend Intelligence
- [ ] Check Facebook Ad Library: how long have they been running ads?
  - Running 1+ year = proof the funnel works; it's profitable
  - Running less than 30 days = possibly testing or failing
- [ ] Check Google: are they bidding on brand keywords?
- [ ] Estimate ad volume (Spy tools: AdSpy, PowerAdSpy, SimilarWeb)

---

## ❌ AUTO-FAIL CONDITIONS

- [ ] ❌ Copy written before competitor analysis completed
- [ ] ❌ Our mechanism is identical to the #1 competitor's mechanism
- [ ] ❌ Our headline makes the same claim as every other player
- [ ] ❌ Offer is worse than competitor offer on every dimension (price, guarantee, bonuses)
- [ ] ❌ We're targeting the same awareness stage as every competitor with same approach
