# Ad Creative Review Checklist
**Agent:** @ad-creative-genius
**Purpose:** Pre-flight validation before launching ads
**Standard:** Frank Kern + Alex Hormozi principles

---

## 🔴 CRITICAL (MUST PASS)

### Thumb-Stopping Power (0-3 seconds)
- [ ] **Pattern interrupt present** (breaks scroll, stops thumb)
- [ ] **Hook in 3 seconds** (video) or headline (static)
- [ ] **High contrast visuals** (4.5:1 ratio minimum)
- [ ] **Mobile-first designed** (60%+ traffic is mobile)

### Hook-Story-Offer Structure
- [ ] **HOOK** (0-3 sec) - Stops scroll, creates curiosity
- [ ] **STORY** (3-30 sec) - Relatable situation + transformation
- [ ] **OFFER** (30-60 sec) - What they get + clear CTA

### Grand Slam Offer (Hormozi)
- [ ] **Dream Outcome clear** (what they want)
- [ ] **Perceived Likelihood high** (proof it works)
- [ ] **Time Delay minimized** ("in 7 days", "instantly")
- [ ] **Effort minimized** ("done-for-you", "plug-and-play")
- [ ] **Value Stack visible** (components + prices)

### Intention-Based Filtering (Kern)
- [ ] **Filters FOR right people** ("If you're serious about X...")
- [ ] **Filters AGAINST wrong people** ("Not interested in quick fixes? Good.")
- [ ] **Polarizing stance** (doesn't try to please everyone)

### Technical Requirements
- [ ] **Captions included** ⚠️ **80% watch muted**
- [ ] **Works without sound** (full message conveyed)
- [ ] **Single clear CTA** (one action, not multiple options)
- [ ] **Mobile tap targets** (44px+ buttons)

---

## 🟡 HIGH PRIORITY

### Testing Protocol
- [ ] **10+ angles created** (problem, solution, transformation, authority, etc.)
- [ ] **3-5 variations per angle** (different creatives, same copy structure)
- [ ] **ONE variable per test** (don't test multiple things)
- [ ] **48-72 hour test window** (sufficient data)

### Ad Types (Format-Specific)

**Static Image:**
- [ ] Text < 20% of image (Meta requirement)
- [ ] Single focal point (not cluttered)
- [ ] CTA button visible
- [ ] Brand/offer clear at glance

**Video (15-30 sec - Awareness):**
- [ ] Hook in first 3 seconds
- [ ] Captions on by default
- [ ] Brand mention at 5 seconds
- [ ] CTA at end

**Video (60-90 sec - Conversion):**
- [ ] Problem shown (first 10 sec)
- [ ] Solution revealed (10-40 sec)
- [ ] Proof/testimonial (40-60 sec)
- [ ] Offer + CTA (60-90 sec)

**UGC-Style:**
- [ ] Raw, authentic feel (not overproduced)
- [ ] Phone camera aesthetic
- [ ] Real person speaking
- [ ] Testimonial or how-to format

### Copy Quality
- [ ] **Benefit-focused** (what they GET, not what it IS)
- [ ] **Specific claims** (numbers, timeframes)
- [ ] **Conversational tone** (sounds human, not corporate)
- [ ] **Urgency element** (limited time, scarcity)

---

## 🟢 BEST PRACTICE

### Advanced Techniques
- [ ] **Pattern interrupt variations tested** (weird, controversial, question)
- [ ] **Multiple angles tested** (pain vs solution vs transformation)
- [ ] **Emotion + logic combo** (feel + think)
- [ ] **Social proof included** (testimonials, user counts, ratings)

### Platform-Specific

**Meta (Facebook/Instagram):**
- [ ] First 3 words hook hard
- [ ] Vertical video for Stories/Reels
- [ ] UTM parameters set
- [ ] Pixel tracking configured

**TikTok:**
- [ ] Native feel (not "ad-like")
- [ ] Trend-aware (current formats)
- [ ] Sound-on optimized (but works muted too)
- [ ] 9-15 seconds optimal

**YouTube:**
- [ ] 15-30 sec (awareness) or 60-90 sec (conversion)
- [ ] Skip consideration (hook before 5 sec)
- [ ] End screen CTA
- [ ] Retargeting audience created

## ❌ COMMON MISTAKES (Auto-Fail)

### Hook Failures
- [ ] ❌ **Boring hook** (blends in with feed)
- [ ] ❌ **Slow start** (doesn't hook in 3 seconds)
- [ ] ❌ **Generic opening** ("Hey guys..." "In this video...")

### Story Failures
- [ ] ❌ **Feature dump** (listing features vs telling story)
- [ ] ❌ **No emotion** (all logic, no feel)
- [ ] ❌ **Too long** (loses attention before CTA)

### Offer Failures
- [ ] ❌ **Weak value stack** (price without value context)
- [ ] ❌ **Multiple CTAs** (confuses - one action only!)
- [ ] ❌ **No urgency** (why act now?)
- [ ] ❌ **Vague CTA** ("Learn More" vs "Get Instant Access")

### Technical Failures
- [ ] ❌ **No captions** (80% watch muted = 80% lost)
- [ ] ❌ **Requires sound** (doesn't work muted)
- [ ] ❌ **Not mobile-optimized** (60%+ traffic)
- [ ] ❌ **Load time > 3 seconds** (loses 50%+)

### Testing Failures
- [ ] ❌ **Testing one ad only** (no winners without losers)
- [ ] ❌ **Testing multiple variables** (can't isolate what works)
- [ ] ❌ **Stopping too early** (< 1,000 impressions)
- [ ] ❌ **Not killing losers fast** (wasting budget)

## 🎯 THE ATTENTION HACKERS' WISDOM

**Frank Kern on Intention:**
> "Your ad should say: 'If you're NOT [ideal customer], keep scrolling. If you ARE [ideal customer], pay attention.' Stop trying to convince the wrong people. Filter them OUT, not in."

**Alex Hormozi on Offers:**
> "Make an offer so good people feel stupid saying no. The formula: increase dream outcome, increase likelihood, decrease time, decrease effort. Then stack so much value that price becomes irrelevant."

**Key Metrics (Target):**

**Awareness:**
- CTR: **2%+**
- CPM: **<$15**
- Hook Rate: **50%+** (3-sec views)

**Conversion:**
- CPA: **<$50** (adjust by niche)
- ROAS: **3x+**
- Conversion Rate: **2%+**

## 📋 QUICK TESTING PROTOCOL

**Week 1-2: Angle Testing**
- 10 angles × $100 each = $1,000
- Goal: Find 2-3 winners (CTR > 2%, CPA < target)

**Week 3-4: Creative Testing**
- 3 angles × 5 creatives = 15 tests
- Budget: $1,500
- Goal: Find scalable winners (ROAS > 3x)

**Week 5+: Scale**
- Budget: $3,000+
- Scale winners only
- Continuous optimization
