# Funnel Review Checklist
**Agent:** @funnel-architect
**Purpose:** Pre-launch funnel validation
**Standard:** Russell Brunson frameworks

---

## 🔴 CRITICAL (MUST PASS)

### Value Ladder Complete
- [ ] **BAIT ($0)** - Free offer defined (lead magnet, challenge, webinar)
- [ ] **FRONTEND ($7-$97)** - Tripwire offer (break-even on ads)
- [ ] **MIDDLE ($97-$997)** - Core offer (profit center)
- [ ] **BACKEND ($1k-$10k)** - High-ticket (max profit per customer)
- [ ] **PREMIUM ($10k+)** - Ultimate offer (optional but ideal)

### Funnel Architecture (Page Flow)
- [ ] **Entry point clear** (ad → landing page)
- [ ] **Each page has ONE purpose** (opt-in, sell, upsell, etc.)
- [ ] **Logical progression** (each step builds on previous)
- [ ] **Thank you pages optimize next step** (not dead ends)
- [ ] **Retargeting audiences planned** (pixel tracking setup)

### Email Sequences (The Glue)
- [ ] **Soap Opera Sequence** (5 emails - nurture/launch)
- [ ] **Sales Sequence** (4-7 emails - open cart)
- [ ] **Indoctrination Sequence** (3-5 emails - new leads)
- [ ] **Each email has clear purpose** (not random)
- [ ] **Emails build on each other** (storyline, not isolated)

### Webinar Funnel (If Applicable)
- [ ] **Perfect Webinar structure** (Intro → 3 Secrets → Pitch)
- [ ] **Registration page optimized** (30-50% opt-in)
- [ ] **Confirmation sequence** (5-7 emails pre-webinar)
- [ ] **Show-up tactics** (reminders, pre-frame)
- [ ] **Replay sequence** (3-5 emails post-webinar)

### Invisible Funnel Elements (Perry Belcher)
- [ ] **Order bump present** (checkbox on checkout - 10-20% AOV lift)
- [ ] **Upsell after purchase** (one-click, 1.5-3x price - 30-50% take rate)
- [ ] **Downsell if declined** (0.5-1x price - 10-20% recovery)
- [ ] **Thank you page has next step** (not dead end)

---

## 🟡 HIGH PRIORITY

### Conversion Optimization

**Landing/Squeeze Pages:**
- [ ] Opt-in rate target: 30-50%
- [ ] Single CTA (not multiple options)
- [ ] Social proof visible
- [ ] Mobile-optimized

**Sales Pages:**
- [ ] Value stack visible
- [ ] Guarantee prominent (60-90 day)
- [ ] Multiple CTAs (every 700 words)
- [ ] Urgency/scarcity present

**Checkout:**
- [ ] Minimal fields (name, email, payment only)
- [ ] Trust badges visible
- [ ] One-page checkout (no multi-step unless necessary)
- [ ] Order bump visible above fold

### Attractive Character
- [ ] **Archetype defined** (Leader, Adventurer, Reporter, Reluctant Hero)
- [ ] **Backstory present** (origin story in emails/webinar)
- [ ] **Voice consistent** (across all pages and emails)
- [ ] **Parables used** (teaching stories)
- [ ] **Polarity shown** (take a stand, have opinions)

### Urgency & Scarcity
- [ ] **Deadline clear** (specific date/time)
- [ ] **Scarcity real** (limited spots, cohorts, seasonal)
- [ ] **Evergreen mechanism** (if evergreen - Deadline Funnel)
- [ ] **Reminder sequence** (countdown emails)

## 🟢 BEST PRACTICE

### Advanced Funnel Types

**Challenge Funnel:**
- [ ] 5-10 day structure
- [ ] Daily training + tasks
- [ ] Community engagement
- [ ] Final webinar/offer

**Application Funnel:**
- [ ] Survey/quiz qualifies
- [ ] Application form (high-ticket only)
- [ ] Calendar booking automated
- [ ] Sales call script prepared

**Product Launch (Jeff Walker PLF):**
- [ ] Pre-launch content (3-4 videos)
- [ ] Launch event (open cart)
- [ ] Close event (deadline)
- [ ] Daily emails during launch

### Email Sequence Quality

**Soap Opera (5 emails):**
1. Set the stage (introduce drama)
2. Build tension (escalate problem)
3. Epiphany (discovery moment)
4. Hidden benefits (unexpected wins)
5. Urgency (deadline, CTA)

**Sales Sequence (7 emails):**
1. Cart open (full offer)
2. Testimonial (social proof)
3. Objection handling (address concerns)
4. Bonus reminder (value stack)
5. Urgency - 48h (countdown starts)
6. Final hours (last chance)
7. LAST CALL - 2h (absolute final)

## ❌ COMMON MISTAKES (Auto-Fail)

### Value Ladder Failures
- [ ] ❌ **No frontend offer** (trying to sell $997 to cold traffic)
- [ ] ❌ **Gaps in ladder** (nothing between $97 and $5,000)
- [ ] ❌ **No ascension path** (one product, nowhere to go)

### Funnel Flow Failures
- [ ] ❌ **Too many steps** (friction kills conversion)
- [ ] ❌ **Confusing navigation** (unclear next step)
- [ ] ❌ **Dead-end thank you pages** (no next step)
- [ ] ❌ **No retargeting** (leaving money on table)

### Email Failures
- [ ] ❌ **Weak subject lines** (low open rates)
- [ ] ❌ **No sequence** (random isolated emails)
- [ ] ❌ **No urgency** (no deadline = no action)
- [ ] ❌ **Too salesy** (pitch every email = unsubscribes)

### Offer Failures
- [ ] ❌ **No order bump** (leaving 10-20% on table)
- [ ] ❌ **No upsell/downsell** (leaving 30-50% on table)
- [ ] ❌ **Weak guarantee** (30-day or less)
- [ ] ❌ **No urgency** (why buy now?)

### Webinar Failures
- [ ] ❌ **Wrong structure** (not Perfect Webinar)
- [ ] ❌ **Too short** (<45 min content)
- [ ] ❌ **Too long** (>90 min total)
- [ ] ❌ **Pitch too early** (before value delivered)
- [ ] ❌ **Low show-up rate** (<30% - need better sequence)

## 🎯 RUSSELL BRUNSON'S FUNNEL WISDOM

> "You're one funnel away from changing everything. The secret isn't finding the perfect funnel - it's finding a proven funnel, hacking it, and making it your own. Funnel hack your way to success."

**The Perfect Webinar Formula:**
1. **Intro** (10 min) - Origin story, build rapport
2. **Secret #1** (10 min) - The vehicle + internal beliefs
3. **Secret #2** (10 min) - Internal beliefs shift
4. **Secret #3** (10 min) - External roadblocks removed
5. **Pitch** (20 min) - Stack, price, bonuses, guarantee, urgency

**Key Metrics (Targets):**
- **Opt-in rate:** 30-50%
- **Webinar show-up:** 30-50%
- **Webinar conversion:** 5-20%
- **Email open rate:** 30-50%
- **Email click rate:** 10-20%
- **Overall funnel CR:** 1-5%
