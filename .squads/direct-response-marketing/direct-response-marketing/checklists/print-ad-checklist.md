# Checklist: Print Ad Quality

**Use when:** Reviewing a print ad (magazine, newspaper, direct mail insert) before production.

---

## Section 1: Headline

- [ ] Headline is the largest element on the page (visual hierarchy)
- [ ] Headline can be read in 3 seconds
- [ ] Headline contains the core promise or key curiosity hook
- [ ] Headline has a subheadline that expands or adds specificity
- [ ] Headline is NOT the company name or logo (common failure mode)

## Section 2: Above-Fold Content (first half of ad)

- [ ] Reader understands what this ad is about in 5 seconds
- [ ] Lead paragraph establishes the promise without requiring full-read
- [ ] First fascination bullet is visible above the fold
- [ ] Image supports the headline (not decorative)
- [ ] Image stops the page turn without requiring context

## Section 3: Body Copy Structure

- [ ] Leads with the problem the reader has (not with the product)
- [ ] Mechanism is explained (WHY this works, not just WHAT it does)
- [ ] At least 3 fascination bullets present
- [ ] Fascination bullets use curiosity gap structure (reveal benefit, withhold method)
- [ ] Proof present: testimonial or data point with specific numbers

## Section 4: Response Mechanism

- [ ] One primary response mechanism (phone / URL / QR code / BRC)
- [ ] CTA is explicit and action-specific ("Call this number to get your free kit")
- [ ] Phone number is large enough to read easily in print
- [ ] URL is short and memorable (not www.company.com/landing-page-for-march-2026)
- [ ] QR code tested before print (leads to correct mobile-optimized page)
- [ ] Business Reply Card (if present): return address, postage-paid, simple form

## Section 5: Specificity

- [ ] Every claim has at least one specific detail (number, name, date, location)
- [ ] Testimonials have full names (no initials or "satisfied customer")
- [ ] Price (if stated) is clearly visible
- [ ] Offer deadline (if present) is specific (date, not "soon")
- [ ] Free offer specifics are clear (what exactly is free, how to get it)

## Section 6: Technical Production

- [ ] Image resolution is 300 DPI minimum
- [ ] Bleed margins set correctly for the publication
- [ ] CMYK color mode (not RGB)
- [ ] Fonts are legible at the print size (minimum 9pt for body copy)
- [ ] White space is used to guide the eye, not fill the page

## Veto Conditions

- VETO if company name/logo is the dominant headline
- VETO if there is no response mechanism with specific instructions
- VETO if image is decorative and doesn't support the headline
- VETO if fascination bullets use features, not benefits
- VETO if body copy starts with "We at [Company]..."
