# Checklist: YouTube Ad Quality

**Use when:** Reviewing a YouTube ad script before production.

---

## Section 1: Hook (First 5 Seconds)

- [ ] Hook is complete in under 5 seconds — no setup required
- [ ] Hook makes skipping feel like a mistake
- [ ] Hook speaks directly to the avatar's situation or desire
- [ ] Hook works without sound (text overlay mirrors spoken words)
- [ ] Hook does NOT start with "Hi, I'm [name] and today I'm going to..."

## Section 2: Structure Integrity (Skippable Ads)

- [ ] First 30 seconds work as a standalone 30-second ad
- [ ] CTA appears by 60 seconds for cold traffic (not only at the end)
- [ ] CTA appears verbally AND visually on screen simultaneously
- [ ] Transition from content to offer is natural (not "okay now for the sales part")
- [ ] Hook → Bridge → Core Argument → Proof → CTA sequence is intact

## Section 3: Script Completeness

- [ ] 90-second version written (primary format)
- [ ] 15-second non-skippable version written
- [ ] 6-second bumper version written
- [ ] 3 hook variations written for A/B testing
- [ ] Each variation tests a different hook type (problem/curiosity/proof/story/contrarian)

## Section 4: Production Notes

- [ ] Visual direction specified for every major section
- [ ] Text overlay specified where spoken words need reinforcement
- [ ] B-roll list is specific (not "show relevant footage")
- [ ] End screen elements defined (subscribe / link / product image)
- [ ] Caption direction included (captions mandatory — 85% viewed without sound)

## Section 5: CTA Elements

- [ ] Verbal CTA is specific ("Click the link below to get your free guide")
- [ ] Screen CTA text is specified (what appears as overlay text)
- [ ] Companion banner text written (for desktop placements)
- [ ] End screen link confirmed matches campaign destination

## Section 6: Traffic Temperature

- [ ] Cold traffic: Ad doesn't assume audience knows the brand
- [ ] Cold traffic: Mechanism is teased but not over-explained (curiosity preserved)
- [ ] Warm/remarketing: Script references what they've seen before
- [ ] Discovery ads: Thumbnail concept and title described (different strategy than in-stream)

## Veto Conditions

- VETO if hook requires context from prior ads or brand knowledge
- VETO if CTA only appears at the end (must appear before 60 seconds for cold traffic)
- VETO if no visual direction provided (production team cannot work blind)
- VETO if 15-second and 6-second versions are missing
- VETO if captions not addressed in production notes
