# Checklist: Direct Mail Package Quality

**Use when:** Reviewing a direct mail letter or package before print and mail.

---

## Section 1: Envelope (The First Sale)

- [ ] Outer envelope has a teaser — gives a reason to open
- [ ] Teaser is not a clever pun — it creates genuine curiosity or urgency
- [ ] Return address is present and legitimate
- [ ] "Urgent" or "Time-Sensitive" teaser is only used if genuinely true
- [ ] Window envelope (if used): envelope window shows personalized element, not just address

## Section 2: Salutation and Lead

- [ ] Letter is addressed to a specific person (not "Dear Friend" for targeted lists)
- [ ] First sentence is the hook — not "I'm writing because..."
- [ ] Lead establishes the problem or promise within the first paragraph
- [ ] Lead is specific (number, situation, or event — not a generic statement)
- [ ] Reader knows what the letter is about within the first 3 paragraphs

## Section 3: Letter Body

- [ ] Letter body follows problem → mechanism → proof → offer structure
- [ ] Paragraphs are short (3-4 lines maximum for readability)
- [ ] Subheadlines guide the eye and carry the story standalone
- [ ] Every page has a Johnson Box or key callout element
- [ ] Proof section comes before price revelation

## Section 4: Offer & Price Presentation

- [ ] Offer is clearly stated (what they get, exactly)
- [ ] Value stack is presented before price
- [ ] Price is specific (not "as low as $X" if there's one price)
- [ ] Price is framed against anchor (daily cost, cost of problem, competitor price)
- [ ] Response deadline is specific (date, not "soon")

## Section 5: P.S. Section

- [ ] P.S. is present (most read element after the headline — Gary Halbert rule)
- [ ] P.S. restates the core offer and deadline in 2-3 sentences
- [ ] P.S. adds urgency or a secondary benefit not in main letter
- [ ] P.P.S. (if present) addresses a key objection or reinforces the guarantee

## Section 6: Response Mechanism

- [ ] Business Reply Card (BRC) has: return address, postage-paid marking, simple fields
- [ ] BRC repeats the offer and key benefits
- [ ] Phone number provided as alternative (not everyone fills out cards)
- [ ] BRC copy sells the act of responding (not just "return this to order")

## Section 7: Technical & List

- [ ] Postage type appropriate for package (first class for testing, bulk for scale)
- [ ] List source and recency documented (older than 12 months needs re-validation)
- [ ] Personalization confirmed for name field
- [ ] Printing specs confirmed with printer (paper weight, fold, insert count)

## Veto Conditions

- VETO if envelope has no teaser (plain envelope dramatically reduces open rate)
- VETO if letter begins with "My name is [X] and I work for [Y]"
- VETO if no P.S. section
- VETO if price is revealed before the value anchor
- VETO if response mechanism requires more than one step
