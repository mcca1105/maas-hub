# Landing Page Review Checklist
**Agent:** @landing-page-psychologist
**Purpose:** Pre-launch page optimization
**Standard:** Cialdini 7 Principles + LIFT Model

---

## 🔴 CRITICAL (MUST PASS)

### Above-the-Fold (3-Second Rule)
- [ ] **Headline visible** (no scroll required)
- [ ] **Headline answers: "What is this?"** (clarity)
- [ ] **Subhead answers: "Is it for me?"** (relevance)
- [ ] **CTA visible** (primary action clear)
- [ ] **Trust signals present** (social proof, badges, authority)

### LIFT Model - Value Proposition (0-10)
- [ ] **Clarity** (3/3) - Crystal clear what this is
- [ ] **Relevance** (3/3) - Obviously for me (target audience)
- [ ] **Value** (4/4) - Compel

ing why I should care

### LIFT Model - Incentive to Action (0-10)
- [ ] **Urgency** (4/4) - Why act NOW (deadline, scarcity)
- [ ] **Specificity** (3/3) - Exactly what happens next
- [ ] **Benefit-focused CTA** (3/3) - "Get Instant Access" not "Submit"

### LIFT Model - Friction (LOWER is better)
- [ ] **Form fields ≤ 3** (each extra field = -10-20% CR)
- [ ] **Page load < 3 seconds** (>3 sec = -25% CR, >5 sec = -50% CR)
- [ ] **Clear path forward** (obvious next step)
- [ ] **No confusing navigation** (single focus)

### LIFT Model - Anxiety (LOWER is better)
- [ ] **Guarantee present** (60-90 day minimum) ⚠️ **CRITICAL**
- [ ] **Testimonials visible** (3+ with photos/video)
- [ ] **Trust badges present** (secure, money-back, payment logos)
- [ ] **Privacy policy linked** (reassurance)
- [ ] **FAQ addresses objections** (common concerns)

## 🟡 HIGH PRIORITY

### Cialdini's 7 Principles

**1. Reciprocity** (Give before asking)
- [ ] Free value offered (lead magnet, sample, training)
- [ ] No ask without give first

**2. Commitment & Consistency** (Small yes → big yes)
- [ ] Micro-commitments used (quiz, survey before opt-in)
- [ ] Progress indicators shown (step 1 of 3)

**3. Social Proof** (We follow the crowd) ⚠️ **HIGH IMPACT**
- [ ] Testimonials with photos/video (3-5 minimum)
- [ ] User counts shown ("Join 10,000+ marketers...")
- [ ] Reviews/ratings visible (if applicable)
- [ ] Trust badges (client logos, media mentions)

**4. Authority** (We defer to experts)
- [ ] Credentials shown (degrees, certifications)
- [ ] Media mentions ("As seen on Forbes, TechCrunch...")
- [ ] Expert endorsements
- [ ] Case studies with data

**5. Liking** (We say yes to those we like)
- [ ] Professional photos (human face builds trust)
- [ ] Personality in copy (not corporate-speak)
- [ ] Origin story (relatable journey)
- [ ] Similarity language ("Fellow entrepreneurs...")

**6. Scarcity** (We want what's scarce) ⚠️ **HIGH IMPACT**
- [ ] Limited spots ("Only 5 left")
- [ ] Deadline timer (countdown visible)
- [ ] Exclusive access ("Not available elsewhere")
- [ ] Seasonal/cohort based ("Q1 enrollment only")

**7. Unity** (We say yes to "us")
- [ ] Shared identity ("For serious marketers who...")
- [ ] Community language ("Join the tribe")
- [ ] Insider terminology

## 🟢 BEST PRACTICE

### Design & UX

**Visual Hierarchy:**
- [ ] F-pattern or Z-pattern layout
- [ ] High contrast CTA button (orange, green, red)
- [ ] White space reduces cognitive load
- [ ] Directional cues (arrows, eye gaze toward CTA)

**Mobile-First (60%+ traffic):**
- [ ] Single-column layout
- [ ] Readable font (16px+ body text)
- [ ] Tap targets ≥ 44px
- [ ] Thumb-friendly CTA (bottom 1/3 of screen)

**Copy Principles:**
- [ ] You-focused (not we/us)
- [ ] Benefit-driven (what they GET)
- [ ] Specific (numbers, timeframes)
- [ ] Active voice ("Get this" not "This can be obtained")
- [ ] Scannable (bullets, subheads, bold)

### Advanced Optimization

**Form Optimization:**
- [ ] Only REQUIRED fields shown (not "nice to have")
- [ ] Inline validation (real-time error checking)
- [ ] Progress indicators (multi-step forms)
- [ ] Auto-fill enabled

**Page Speed:**
- [ ] Images compressed (WebP format)
- [ ] Lazy loading (below-fold content)
- [ ] Minified CSS/JS
- [ ] CDN used (CloudFlare, etc.)

**Heatmap Insights (if available):**
- [ ] Clicked CTA (click tracking)
- [ ] Scroll depth (how far users go)
- [ ] Dead zones (ignored sections)
- [ ] Rage clicks (frustration points)

## ❌ COMMON MISTAKES (Auto-Fail)

### Above-the-Fold Failures
- [ ] ❌ **Headline not visible** (requires scroll)
- [ ] ❌ **Generic headline** ("Welcome!" vs specific benefit)
- [ ] ❌ **Vague value prop** (what IS this?)
- [ ] ❌ **CTA not visible** (hidden below fold)

### Friction Failures
- [ ] ❌ **Too many form fields** (>5 fields = massive drop-off)
- [ ] ❌ **Slow load time** (>3 sec = lose 50%+)
- [ ] ❌ **Multiple CTAs** (confuses - one action only!)
- [ ] ❌ **Complicated process** (unclear next step)

### Anxiety Failures
- [ ] ❌ **No guarantee** (or weak 30-day)
- [ ] ❌ **No testimonials** (or fake-looking stock photos)
- [ ] ❌ **No trust badges** (looks sketchy)
- [ ] ❌ **No social proof** (no one else is doing this?)

### Mobile Failures
- [ ] ❌ **Not mobile-optimized** (60%+ traffic lost)
- [ ] ❌ **Tiny text** (<14px on mobile)
- [ ] ❌ **Small buttons** (<44px = hard to tap)
- [ ] ❌ **Horizontal scroll** (bad UX)

### Design Failures
- [ ] ❌ **Low contrast** (CTA blends in)
- [ ] ❌ **Cluttered** (too much happening)
- [ ] ❌ **No white space** (overwhelms)
- [ ] ❌ **Generic stock photos** (fake feeling)

## 🎯 ROBERT CIALDINI'S WISDOM

> "Every element on a page is a micro-decision. Our job is to remove friction and amplify desire until 'Yes' is the only logical answer. This isn't manipulation - it's understanding human psychology and designing experiences that guide transformation."

**The LIFT Formula:**
```
Conversion = (Value Prop + Incentive - Friction) - Anxiety

Maximize: Value Proposition + Incentive
Minimize: Friction + Anxiety
```

**Key Metrics (Targets):**
- **Conversion Rate (warm traffic):** 20-30%
- **Conversion Rate (cold traffic):** 10-15%
- **Bounce Rate:** <40%
- **Time on Page:** 2+ minutes
- **Scroll Depth:** 60%+ reach bottom

## 📋 A/B TESTING ROADMAP

**Phase 1: Big Swings** (10-50% lift potential)
1. Headline variations (5 tests)
2. Hero image/video (image vs video)
3. CTA copy & color (benefit-focused, urgent)

**Phase 2: Elements** (5-20% lift potential)
4. Form fields (reduce by 1-2)
5. Guarantee placement (above-fold + bottom)
6. Social proof format (video vs text testimonials)

**Phase 3: Polish** (2-10% lift potential)
7. Button text ("Get Access" vs "Start Now")
8. Color scheme (trust vs urgency colors)
9. Micro-copy (subheads, bullets)

**Testing Rules:**
- ✅ ONE variable at a time
- ✅ 95% statistical significance
- ✅ Minimum 1,000 conversions per variant
- ✅ Run until significance (not time-based)
