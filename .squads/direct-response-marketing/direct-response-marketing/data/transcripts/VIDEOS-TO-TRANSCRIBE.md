## Extraction Process

### Using Apify Actor: pintostudio/youtube-transcript-scraper

**Input Required:**
```json
{
  "videoUrl": "https://youtube.com/watch?v=...",
  "lang": "en",
  "format": "text"
}
```

**Expected Output:**
- Full video transcript with timestamps
- Speaker identification (if available)
- Cleaned text format

### Manual Alternative (if Apify unavailable):

1. **YouTube Auto-Transcript:**
   - Open video → Click "..." → Show Transcript
   - Copy full transcript
   - Format with timestamps

2. **Professional Service:**
   - Use Rev.com or Otter.ai
   - Upload video URL
   - Get cleaned transcript

3. **YouTube-DL + Whisper:**
   ```bash
   youtube-dl [URL] -x --audio-format mp3
   whisper audio.mp3 --model large
   ```
