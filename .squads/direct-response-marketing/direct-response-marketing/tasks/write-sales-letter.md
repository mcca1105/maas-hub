# Task: Write Long-Form Sales Letter

**Agent:** gary-halbert
**Tier:** 2
**Type:** Creation
**Elicit:** true

## Purpose
Write a complete, high-converting long-form sales letter using Halbert's proven structure.

## Prerequisites
- Starving crowd defined (Recency, Frequency, Unit of Sale)
- Awareness stage identified (from eugene-schwartz)
- Research complete (prospect language, desires, objections)

## Inputs Required
- Target market definition (the starving crowd)
- Awareness stage (1-5)
- Product details and proof inventory
- Offer details (price, bonuses, guarantee)
- 25 headline candidates (or request to generate)

## Execution Steps

### Step 1: Headline (Write 25, Choose Best)
Apply Halbert's 25-headline rule before selecting.
Headline must:
- STOP them
- Make a specific promise or create curiosity
- Target the avatar specifically (not everyone)

### Step 2: Salutation and Opener
- Address ONE person
- Qualify the reader (make them self-select)
- Example: 'If you're a business owner frustrated with...'

### Step 3: Story Hook
True story that creates identification and builds trust:
- Before state (struggle)
- Discovery moment
- Transformation
- Why you're sharing this

### Step 4: Problem Agitation
- Validate the problem (they're not alone)
- Pile on consequences
- Make not solving it feel MORE painful than solving it

### Step 5: Solution Introduction
- Bridge: 'That's why I developed...'
- Introduce product AFTER they feel the problem

### Step 6: Proof Section
- Testimonials (specific: name, location, measurable result)
- Case studies
- Statistics/research
- Expert validation

### Step 7: Fascination Bullets (25-50 minimum, use best 15-20)
Each bullet:
- Creates a knowledge gap
- Teases specific content inside
- Is counterintuitive or surprising
- Avoids being a feature description

### Step 8: Offer Block
- Product what's included
- Bonus stack (with values)
- Price (anchored)
- Guarantee (full risk reversal)
- Urgency/scarcity

### Step 9: Close and CTA
- Summarize transformation
- Single CTA
- Make action easy

### Step 10: P.S. (MANDATORY)
The P.S. must contain:
- Biggest benefit
- The offer summary
- Deadline reminder
- CTA link

## Quality Standards
- 25+ headlines written before selection
- P.S. is mandatory and complete
- All testimonials have name + location + specific result
- Bullets are fascinations, not feature descriptions
- Scarcity is real with explanation

## Veto Conditions
- VETO if fewer than 10 headline options generated
- VETO if P.S. is missing
- VETO if testimonials are unnamed or non-specific
- VETO if bullets describe features rather than create curiosity
- VETO if no urgency mechanism present

## Completion Criteria
- [ ] 25 headline candidates written
- [ ] Best headline selected with rationale
- [ ] All structural sections present
- [ ] 25+ fascination bullets written, best 15-20 selected
- [ ] P.S. complete with offer + benefit + deadline
- [ ] Proof section has 3+ specific testimonials
- [ ] Letter reads like a friend sharing a discovery
