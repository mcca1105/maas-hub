# Direct Response Marketing - Coding Standards

## Copy Writing Standards

### Headlines
- Use power words and curiosity gaps
- Keep under 20 words for digital
- Test minimum 3 variations
- Include benefit or transformation

### Body Copy
- Short paragraphs (2-3 sentences max)
- Use bullet points for scannability
- Include social proof every 200-300 words
- End sections with micro-commitments

### CTAs
- Action-oriented verbs (Get, Claim, Access, Start)
- Create urgency (limited time, scarcity)
- Repeat CTA every 500-700 words
- Use contrasting colors for buttons

## VSL Script Standards

### Structure
1. **Pattern Interrupt** (0-15 seconds)
2. **Hook** (15-45 seconds)
3. **Story** (3-8 minutes)
4. **Mechanism** (2-5 minutes)
5. **Offer** (3-5 minutes)
6. **Close** (1-2 minutes)

### Timing
- Total length: 15-25 minutes optimal
- Pause every 3-5 minutes for breathing room
- CTA placement: 70%, 85%, 95%, 100%

### Visual Guidelines
- Text overlays for key points
- B-roll every 30-45 seconds
- Testimonials at 40% and 75%
- Guarantee section at 85%

## Ad Creative Standards

### Static Ads
- **Hook:** First 3 words must stop scroll
- **Image:** Contrast ratio > 4.5:1
- **Text:** < 20% of image area (Meta)
- **CTA:** Single, clear action

### Video Ads
- **Length:** 15-30 seconds (awareness), 60-90 seconds (conversion)
- **Captions:** Always included
- **Sound:** Works with and without audio
- **First 3 seconds:** Critical - must hook

### Testing Protocol
- Minimum 3 angles per campaign
- 5-10 creative variations per angle
- Test one variable at a time
- 48-72 hours per test cycle

## Funnel Page Standards

### Above the Fold
- Headline visible without scroll
- Supporting subhead
- Single CTA button
- Hero image/video
- Trust badges

### Form Design
- Minimum required fields only
- Progress indicators for multi-step
- Inline validation
- Mobile-first approach

### Loading Speed
- < 3 seconds on 4G
- Lazy load below fold
- Optimize images (WebP)
- Minify CSS/JS

## Analytics Standards

### Tracking
- UTM parameters on all links
- Event tracking for all CTAs
- Scroll depth tracking
- Heatmap on all landing pages

### KPIs to Monitor
- **Traffic:** CPM, CPC, CTR
- **Engagement:** Bounce rate, time on page, scroll depth
- **Conversion:** CR%, CPL, CPA
- **Revenue:** AOV, LTV, ROAS

### Reporting Frequency
- **Daily:** Ad performance, spend
- **Weekly:** Funnel metrics, conversions
- **Monthly:** LTV cohorts, attribution analysis

## Naming Conventions

### Files
- `headline-v1-curiosity.txt`
- `vsl-script-product-name-v2.pdf`
- `ad-creative-angle-pain-v3.png`

### Campaigns
- `{product}_{funnel-type}_{traffic-source}_{date}`
- Example: `course_webinar_fb_2026-02`

### Assets
- `{type}_{product}_{variant}_{version}`
- Example: `cta_button_red_v2.png`

## Version Control

### Copy Versions
- v1, v2, v3... for major iterations
- v1.1, v1.2... for minor tweaks
- Always keep original (v0)

### A/B Test Naming
- Control: `{name}-control`
- Variant: `{name}-var-a`, `{name}-var-b`

## Quality Checklist

### Before Launch
- [ ] Spell check and grammar review
- [ ] Mobile responsive test
- [ ] All links working
- [ ] Tracking pixels installed
- [ ] Thank you page configured
- [ ] Email sequences ready

### Post-Launch
- [ ] Conversion tracking verified
- [ ] No console errors
- [ ] Load time < 3 seconds
- [ ] Forms submitting correctly
- [ ] Payment processing working
