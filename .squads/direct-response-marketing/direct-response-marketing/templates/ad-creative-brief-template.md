## 🧠 PSYCHOLOGICAL TRIGGERS (Cialdini Principles)

Which persuasion principles will this ad leverage?

- [ ] **Reciprocity:** Free value upfront (lead magnet, insight)
- [ ] **Commitment:** Micro-commitment (quiz, survey before offer)
- [ ] **Social Proof:** Testimonials, user counts, reviews
- [ ] **Authority:** Credentials, media mentions, expert status
- [ ] **Liking:** Relatability, personality, shared values
- [ ] **Scarcity:** Limited availability (time, quantity, access)
- [ ] **Unity:** Shared identity ("for serious marketers like us")

## ✅ PRE-FLIGHT CHECKLIST

Before launching this ad, confirm:

**Creative Quality:**
- [ ] Stops scroll in first 0.3 seconds (pattern interrupt)
- [ ] Hook clearly visible (first 3 seconds for video, headline for image)
- [ ] Visual is high-quality (no blurry, pixelated images)
- [ ] Text overlay is readable on mobile
- [ ] Captions on entire video (if video)
- [ ] Works without sound
- [ ] CTA is crystal clear

**Messaging:**
- [ ] Speaks directly to target avatar
- [ ] One clear angle (not trying to say too much)
- [ ] Benefit-focused (not feature-focused)
- [ ] Creates curiosity or emotion
- [ ] Filters FOR ideal customer
- [ ] Filters AGAINST wrong customer

**Offer:**
- [ ] Clear what they're getting
- [ ] Value is obvious
- [ ] Next step is frictionless
- [ ] Urgency/scarcity present (if applicable)

**Technical:**
- [ ] Landing page is live and working
- [ ] Forms are tested
- [ ] UTM tracking is set up
- [ ] Conversion pixels are firing
- [ ] Mobile-optimized (test on phone)
- [ ] Load time under 3 seconds

**Compliance:**
- [ ] Follows platform ad policies (Meta, Google, TikTok)
- [ ] No prohibited content
- [ ] Claims are defensible
- [ ] Testimonials have disclaimers (if required)
- [ ] Text in image is under 20% (if Meta)
