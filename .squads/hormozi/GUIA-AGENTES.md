## 8 Mental Models Hormozi

| Modelo | Fórmula |
|--------|---------|
| **Value Equation** | `(Dream × Likelihood) ÷ (Tempo + Esforço)` |
| **Grand Slam Offer** | "Oferta tão boa que as pessoas se sentem burras dizendo não" |
| **Core Four** | Warm, Content, Cold, Paid (4 canais) |
| **Rule of 100** | 100 ações diárias = crescimento garantido |
| **LTV:CAC 3:1** | Benchmark de saúde financeira |
| **Framework-First** | Converter problemas em frameworks sistemáticos |
| **Metrics as Reality** | Se não é mensurável, não existe |
| **More, Better, New** | Sequência de crescimento (mais primeiro, novo por último) |

## Como Usar

### 1. Ativar o Squad Hormozi

```bash
@hormozi
```

Isso carrega o chief (orquestrador) que roteia para o agente certo.

### 2. Comandos Disponíveis no Chief

```
*help                  → Ver todos os comandos
*diagnose              → Value Equation diagnostic
*health-check          → Quick LTV:CAC check
*squad                 → Listar todos 15 agentes
*mental-models         → Explicar 8 mental models
*antipatterns          → Detectar antipatterns
*route [topic]         → Rotear para especialista
*guide                 → Onboarding interativo

Diretos:
*offers                → Ir para hormozi-offers
*leads                 → Ir para hormozi-leads
*models                → Ir para hormozi-models
*hooks                 → Ir para hormozi-hooks
*ads                   → Ir para hormozi-ads
*pricing               → Ir para hormozi-pricing
*copy                  → Ir para hormozi-copy
*launch                → Ir para hormozi-launch
*retention             → Ir para hormozi-retention
*advisor               → Ir para hormozi-advisor
*audit                 → Ir para hormozi-audit
*closer                → Ir para hormozi-closer
*scale                 → Ir para hormozi-scale
*workshop              → Ir para hormozi-workshop
*content               → Ir para hormozi-content
```

### 3. Acessar Agente Específico

```bash
@hormozi:offers        # Ir direto para hormozi-offers
@hormozi:leads         # Ir direto para hormozi-leads
@hormozi:scale         # Ir direto para hormozi-scale
```

### 4. Rodar um Workflow

```bash
# Dentro de @hormozi ou @hormozi:offers
*workflow wf-grand-slam-offer

# Ou via task direto
*task create-grand-slam-offer
```
